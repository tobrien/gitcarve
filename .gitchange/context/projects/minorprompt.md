### MinorPrompt

MinorPrompt is a tool that helps you write better commit messages and release notes.

MinorPrompt is developed by Tim O'Brien.

MinorPrompt is a command-line tool.

MinorPrompt is a Node.js project.

MinorPrompt is open source software.

MinorPrompt is free software.

MinorPrompt is licensed under the Apache 2.0 license.

MinorPrompt is available at https://github.com/tobrien/minorprompt
